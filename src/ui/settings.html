提示词优化 LLM

“复制图片连接”只复制 API Key 和地址；提示词优化沿用图片生成代理。调用方式和模型仍需单独确认。
开启后，存在参考图时会把参考图发送给提示词优化模型。
如果模型不支持图片输入，会直接报错并提示更换多模态模型或关闭此选项。
